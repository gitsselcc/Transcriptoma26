# Transcriptoma26

Repositorio de Transcriptomica y Alineamientos

Los datos analizados provienen del proyecto Tabula Muris Senis (GEO: GSE132040), un atlas transcriptómico del envejecimiento en ratón.

• Tipo de datos: RNA-seq bulk

• Especie: Mus musculus

• Tejido: BAT (Brown Adipose Tissue)

• Condición: machos

• Edades: 3 y 27 meses

Los SRA especificos de mi asignacion se encuentran en SraRunTable.csv
